[git 공부]
git이란 로컬에서 관리되는 버전 관리 시스템
github는 공유가 필요한 소스를 업로드하고, 내려받을 수 있는 원격저장소(클라우드)

@ git 사용하기 위한 기본 익히기
* git 계정 및 repository 생성 
* 명령어
commit : git으로 로컬에 코드 저장 -> git commit -m 'test'
push : github에 업로드하여 다른 이들과 공유 혹은 개인소스 버전관리 -> git push origin master
pull : github에서 다운로드 받아 사용 -> git pull origin master
